# 🪑 GestorMesas

Aplicación web para la gestión visual de mesas en una cafetería o restaurante. Permite ver la distribución de mesas, agregar nuevas, editarlas y organizar su disposición.

## 🚀 Tecnologías utilizadas

- [React](https://reactjs.org/)
- [TypeScript](https://www.typescriptlang.org/)
- [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [ShadCN/UI](https://ui.shadcn.com/)
- [Lucide Icons](https://lucide.dev/)
- [UUID](https://www.npmjs.com/package/uuid)

## 🛠 Requisitos

- **Node.js 18.x recomendado**  
- **Vite v5.x** (⚠️ Vite 6 o 7 puede generar errores con algunas dependencias)

## 🧑‍💻 Instalación local

```bash
git clone https://github.com/LippigGian/GestorMesas.git
cd GestorMesas
npm install
npm run dev
